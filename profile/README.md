## 🏭 About Gas Town Hall

**Gas Town Hall** is the official community and documentation hub for **Gas Town** — an open-source AI agent orchestration layer that helps you build with AI coding agents at scale.

> *Building community for Gas Town, your AI agent orchestrator.*
>
> Gas Town links thousands of AI-powered rigs together in a trust network, letting you build software really, really fast. Track accountability, measure quality, and scale your AI-assisted engineering workflows.
>

>
> ## 🔧 What is Gas Town?
>
> Gas Town is an orchestration layer for AI coding agents. It lets you:
>
> - 🤖 **Orchestrate AI agents** — Works with Claude Code, Codex, Gemini, and more
> - - 🌐 **Scale without limits** — Link thousands of Gas Towns together in the Wasteland trust network
>   - - 📊 **Track accountability** — Measure quality and output across all your AI coding workflows
>     - - ⚡ **Build fast** — Deploy changes and contributions at unprecedented speed
